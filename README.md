## Ejecutar localmente

**Requisitos previos:** Node.js

1. Instalar dependencias:
   `npm install`
2. Configurar la variable `GEMINI_API_KEY` en el archivo `.env.local` con tu clave de API de Gemini
3. Ejecutar la aplicación:
   `npm run dev`
